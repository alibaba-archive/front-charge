「前端充电站」是由阿里巴巴淘系前端团队发起的前端技术分享会

希望能够在不同的城市和不同的同学通过前端会友、一起分享技术的乐趣。

点此报名

缺月 @淘宝前端智能化团队

面向前端解决业务问题的智能化算法平台

前端作为离用户最近的一环，在业务中有大量的数据分析需求，同时，现有的机器学习生态和前端生态割裂，很难直接融入到前端工程当中去，也不能直接在端上使用，目前业界也缺乏前端生产环境的机器学习可持续集成和交付能力，分享话题将会讲述淘宝前端在智能化上的实践和落地，同时如何通过自研框架，将机器学习的能力运用到前端业务当中去。

涂志坚 @亚信科技

复杂场景下生成小程序海报

在前端开发中，我们一般都会遇到图片生成，然后预览或者保存到本地的需求。在 Web 开发中，我们可以通过 html2canvas 解决类似场景的问题。由于小程序和浏览器的执行环境存在差异，我们并不能直接沿用之前的方案，本次演讲将分享通过前端技术，在小程序中生成复杂海报的解决方案。

晟怀 @淘宝前端架构团队

基于 Flutter 的高性能 Web 渲染引擎

在跨端渲染这一前端的传统领域，Flutter 凭借它的自绘渲染管线，可以说是风光无两；然而 Flutter 使用的 Dart 语言和 Widget 体系却给传统的前端开发增加了非常高的门槛，生态体系完全割裂造成研发成本的提升。基于这一背景，Kraken 在完全遵守 W3C 标准的基础上，使用 Flutter 的渲染管线，做到了自绘的高性能渲染。本次演讲将分享 Kraken 在一线实践与思考。

沈洪梁 @有赞

有赞店铺装修实践

有赞店铺装修，是一款可以由商家运营自主组装页面的工具，通过可视化搭建的方式，帮商家快速创建店铺主页、商品活动页等，分享话题讲述了有赞在店铺装修相关的实践经验，包括设计理念、实践过程和思考总结，以及对未来的规划。
